# Full-Book Ingestion Protocol

## Phase 1: Inventory

Identify all manuscript units:
- chapter headings in `manuscript.md`, or
- all files inside `/book/` if split by chapter.

Create `outputs/01_coverage_audit.md` before making judgments.

## Phase 2: Neutral Reading Memory

For each unit, record:
- what literally happens
- character state changes
- relationship state changes
- motifs/images/jokes/phrases introduced or repeated
- questions created for later payoff
- scenes likely to be recontextualized later

Use `memory/` files as working memory.

## Phase 3: Retrospective Re-evaluation

After the entire book is read, identify:
- early scenes whose meaning changes later
- delayed emotional payoffs
- callbacks that become stronger in hindsight
- apparent weaknesses that become intentional design
- apparent strengths that weaken after later context

## Phase 4: Critic-Agent Review

Only after Phases 1-3, run five independent critic lenses:

- Wood Lens: consciousness, sentence rhythm, moral perception
- Kakutani Lens: authority, pacing, verdict, earned ambition
- Sehgal Lens: anti-reduction, mystery, over-explanation
- Emre Lens: form, desire, institutions, intellectual architecture
- Chu Lens: hostile reading, aesthetic alibis, hidden politics

Each critic must cite specific chapter/scene evidence.

## Phase 5: Debate

The critics read one another only after independent reviews.

Debate questions:
1. What did another critic miss?
2. Which apparent flaw is actually structural design?
3. Which defended choice still fails for a reader?
4. Which issue appears across multiple lenses?
5. Which critique is only taste/preference?
