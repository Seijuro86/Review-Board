# Output Format

## `/outputs/final_roundtable_review.md`

```markdown
# Final Roundtable Review

## Executive Verdict

## Full-Book Understanding

## Consensus Strengths

## Consensus Weaknesses

## Recursive Structure Analysis

## Character Arc Analysis

## Prose / Form / Style Analysis

## Emotional Payoff Analysis

## Reader Reception Forecast

## Revision Priorities
```

## `/outputs/dissent_reports.md`

```markdown
# Dissent Reports

## Wood Dissent

## Kakutani Dissent

## Sehgal Dissent

## Emre Dissent

## Chu Dissent
```

Only include true dissents. Do not fabricate disagreement if a critic agrees.

## `/outputs/actionable_revision_plan.md`

```markdown
# Actionable Revision Plan

## Fix First
Major structural or emotional issues that affect the whole book.

## Fix Second
Character, pacing, payoff, and thematic clarity issues.

## Fix Later
Line-level, scene-level, or polish issues.

## Do Not Break
Strengths the author should preserve during revision.
```
